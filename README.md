# Outside Localhost

A modern portfolio website showcasing **Outside Localhost activities** like workshops, meetups, community initiatives, and speaking engagements.

> **⚠️ Disclaimer:** This is NOT a technical project portfolio. For technical work, projects, and coding expertise, please reach out through the contact section.

## Features

- **Responsive Design** - Built with Tailwind CSS for mobile-first responsive layouts
- **Multiple Sections** - Hero, About, Experience, Workshops, Offerings, Resources, Contact, and Footer
- **Type-Safe** - Written in TypeScript for better code reliability
- **Fast Build** - Powered by Vite for rapid development and optimized builds

## Tech Stack

- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Vite** - Build tool and dev server
- **ESLint** - Code quality
